# Datenmodell

## Lagerplatz

Ein Lagerplatz besteht aus:

- Name
- Beschreibung
- Hintergrundbild
- Maßstab
- Stromanschlüsse
- Wasseranschlüsse
- Objekte
- Checklisten

---

## Objekt

Jedes Objekt besitzt:

- ID
- Typ
- Name
- Position X
- Position Y
- Rotation
- Breite
- Höhe
- Farbe
- Sichtbar
- Notizen

---

## Checkliste

- Titel
- Kategorie
- Verantwortlicher
- Fällig
- Status
- Priorität

---

## Strom

- Verteiler
- Anschluss
- Leistung
- Kabel

---

## Wasser

- Anschluss
- Schlauch
- Verbraucher
