# 🏕️ CampPlanner

Der digitale Zeltlager-Planer.

## Funktionen

- 🗺️ Interaktiver Lageplan
- 🏕️ Drag & Drop von Zelten
- ⚡ Stromplanung
- 🚰 Wasserplanung
- ✅ Aufbau- und Abbauchecklisten
- 📅 Zeitplanung
- 📄 Dokumente (OneDrive)
- 📱 Optimiert für Smartphone und Tablet
- 🌐 Offline nutzbar (PWA)
